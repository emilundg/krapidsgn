# KrapiDsgn.

The main project for krapidsgn.com

## Build Setup

### install dependencies
``` bash
npm install
```

## Serve
``` For development ```

``` bash

# serve with hot reload at localhost:8080
npm run dev
```

``` For production ```

``` bash

# build for production with minification
npm run build

# serve at localhost:5000
npm run start
```
